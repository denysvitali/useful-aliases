# Useful Aliases
A collection of useful aliases to be faster and more efficient

## Contents

- Git
- Docker

### Git
#### gpom
Push origin to master  
`gpom`
#### gpo
Push current branch to origin  
`gpo`
#### gcom
Commit with a message  
`gcom "My message"`
#### gadd
Add file  
`gadd README.md`
#### gagh
Adds the GitHub repository as origin  
`gagh username/repository` (adds git@github.com:/username/repository.git as origin)

### Docker
#### dockerX11run
Run containerized apps with a X11 environment  
For a list of GUI apps, look at [Jessie Frazelle](https://hub.docker.com/u/jess/)'s profile on Docker Hub or [her awesome Dockerfiles on GitHub](https://github.com/jessfraz/dockerfiles)
`dockerX11run jess/tor-browser`
